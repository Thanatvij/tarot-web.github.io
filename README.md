## ✨ ฟีเจอร์

### ระบบดูดวง
- 🃏 **ไพ่ทาโรต์ครบ 78 ใบ** — Major Arcana 22 ใบ + Minor Arcana 56 ใบ (Wands/Cups/Swords/Pentacles)
- 🎴 **ภาพไพ่จริง** — Rider-Waite-Smith Tarot โดย Pamela Colman Smith (1909, Public Domain)
- 🌓 **เรียงไพ่เป็นพัดครึ่งวงกลม** — สวยขลัง ตามแบบหมอดูดั้งเดิม ด้วย CSS transforms
- 🔄 **ไพ่กลับหัว** — 30% โอกาสได้ไพ่กลับหัว พร้อมความหมายเฉพาะ
- 📱 **Responsive Design** — รองรับทุกอุปกรณ์ ด้วย Tailwind CSS + Custom CSS
- 🎯 **เลือกหมวดได้ 5 หมวด** — ความรัก / การเรียน / การทำงาน / การเงิน / ดวงชะตาภาพรวม
- 🔮 **ระบบตรวจจับคำถาม** — แยกประเภทคำถามอนาคต vs ทั่วไป ให้ spread ที่เหมาะสม
  - **คำถามทั่วไป**: เจ้าชะตา → สถานการณ์ → บทสรุป
  - **คำถามอนาคต**: อดีต → ปัจจุบัน → อนาคต
- 📖 **คำพยากรณ์รวม** — วิเคราะห์ภาพรวมจากจำนวน Major Arcana และชุดไพ่ที่ปรากฏซ้ำ
- 🎨 **ดีไซน์ขลังลึกลับ** — โทนม่วง+ทอง พร้อม animation, ดาวระยิบระยับ, พระจันทร์เรืองแสง
- ✨ **Animation สับไพ่สุดอลังการ** — Riffle shuffle effect พร้อม particle effects
- ✂️ **ฟีเจอร์ตัดไพ่** — ส่งพลังของคุณเข้าสู่สำรับไพ่ เลือกตัดไพ่ได้ตามใจฉัน
- 🎯 **ปุ่มตัดไพ่ตรงกลาง** — วางตำแหน่งอย่างแม่นยำตรงกลางสำรับไพ่ (54% desktop, 62% mobile)
- 📐 **3D card stack** — ภาพกองไพ่ 3 มิติ สวจงามด้วย pseudo-elements และ layered shadows

### สื่อสารและแชร์
- 📤 **บันทึกเป็นภาพ** — สร้างภาพผลการดูดวง 1080x1080px สวยงาม
- 📱 **Native Share (iOS/Android)** — แชร์ผลลัพธ์ได้ทันที
- 💻 **Desktop Download** — ดาวน์โหลดไฟล์ PNG คมชัด

## 🚀 วิธีใช้งาน

### เปิดในเครื่อง (Local Development)

```bash
# วิธีที่ 1: เปิดตรงๆ (ต้องต่อเน็ตเพื่อโหลดรูปไพ่จาก CDN)
open index.html

# วิธีที่ 2: รัน server (แนะนำ)
python3 -m http.server 8000
# จากนั้นเปิด http://localhost:8000
```

### วิธีดูดวง

1. กดปุ่ม **"เริ่มเปิดไพ่"**
2. เลือกหมวดที่ต้องการพยากรณ์ (ความรัก / การเรียน / การทำงาน / การเงิน / ดวงชะตาภาพรวม)
3. ระบุคำถามเพิ่มเติม (ถ้ามี) — ไม่จำเป็น
   - **💡 ระบบจะตรวจจับว่าเป็นคำถามเชิงอนาคตหรือไม่** เพื่อเลือก spread ที่เหมาะสม
4. **ดู Animation สับไพ่** — สับไพ้อย่างอลังการด้วย riffle shuffle effect
5. **เลือกตัดไพ่ (Optional)** — กดปุ่ม "ตัดไพ่" เพื่อส่งพลังของคุณเข้าสู่สำรับ
   - เลือกกองบนหรือกองล่างเพื่อวางบนสุด
6. **เลือกไพ่ 3 ใบจากพัดครึ่งวงกลม** — คลิกที่ไพ่ที่สะดุดตา
   - มีโอกาส 30% ที่ไพ่จะเป็น **กลับหัว** (สีแดง หมุน 180°)
7. รับคำพยากรณ์จากจักรวาล ✨
8. กด **"บันทึกและแชร์"** เพื่อบันทึกภาพผลลัพธ์

## 🛠 เทคโนโลยี

### Core Technologies
- **HTML5** — โครงสร้างหน้าเว็บ
- **CSS3** — ดีไซน์ + animation 3D + พัดครึ่งวงกลม (CSS transforms)
- **Tailwind CSS** — Responsive utilities (ผ่าน CDN, ไม่ต้อง build)
- **Vanilla JavaScript** — ไม่ใช้ framework
- **Google Fonts** — Cinzel, Noto Serif Thai, Sarabun
- **jsDelivr CDN** — โหลดรูปไพ่ Rider-Waite (Public Domain) จาก npm package `tarot-card-img`

### ไม่มีอะไรเหล่านี้
❌ ไม่ใช้ external API  
❌ ไม่ใช้ localStorage  
❌ ไม่ต้อง build step  
❌ ไม่ต้อง install dependencies  
✅ ทำงาน offline ได้เกือบหมด (ยกเว้นรูปไพ่ที่โหลดจาก CDN)

## 🎴 ระบบการอ่านไพ่

### การเรียงไพ่ (Card Spread)
เรียงไพ่แบบ **พัดครึ่งวงกลม (Semi-Circular Fan)** โดยใช้ CSS transforms:
- ไพ่ 78 ใบกระจายเป็นพัดมุม 160°
- ไพ่แต่ละใบมีการคำนวณ position ด้วย JavaScript
- Responsive radius — ปรับขนาดตามหน้าจอ (220px → 160px → 115px → 95px)
- **ไพ่กลับหัว** — สุ่ม 30% เมื่อสร้างพัดไพ่

### ระบบตรวจจับประเภทคำถาม
ระบบจะตรวจจับคำในคำถามของคุณ เพื่อเลือก spread ที่เหมาะสม:

**คำหลักที่ตรวจจับ:**
- จะ, ไหม, มั้ย, อนาคต, หน้า, ต่อไป, รุ่ง, เป้า, ทัน, ได้, เป็นยังไง, ผล, รอด, สำเร็จ, ชนะ, ไปต่อ, แต่ง, รับ

**Spread ที่ใช้:**
1. **อดีต - ปัจจุบัน - อนาคต** (สำหรับคำถามเชิงอนาคต)
   - อดีต: สิ่งที่ผ่านมา
   - ปัจจุบัน: สถานการณ์ปัจจุบัน
   - อนาคต: แนวโลค 1-3 เดือน

2. **เจ้าชะตา - สถานการณ์ - บทสรุป** (สำหรับคำถามทั่วไป)
   - เจ้าชะตา: ตัวคุณและพลังงาน
   - สถานการณ์: สิ่งที่เผชิญ
   - บทสรุป: คำแนะนำและผลลัพธ์

### ไพ่กลับหัว (Reversed Cards)
- เกิดขึ้นสุ่ม 30% เมื่อสับไพ่
- สีแดง และมีข้อความ "(กลับหัว)"
- ใช้ **ความหมายตรงกันข้าม** จากไพ่ตัวตรง
- สะท้อนพลังงานที่ถูกบดบัง หรือสิ่งที่ต้องระวัง
